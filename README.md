# SimoraTodo 

A simple productivity app to manage tasks and reminders, built with React + Tailwind CSS.

##  Tech Stack
- React.js
- Tailwind CSS
- LocalStorage

##  Features
- Add and remove tasks
- View reminders on a calendar
- Save tasks in your browser (no login needed)
